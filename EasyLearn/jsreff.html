Introduction

JavaScript is a lightweight, interpreted programming language used to create interactive effects within web browsers.

Basic Syntax

// Example
console.log("Hello World!");
let x = 10;

Variables

JavaScript variables can be declared using var, let, or const. 

let name = "John";
const age = 25;
var city = "New York";
console.log(name, age, city);

Functions

function greet(name) {
  return "Hello " + name;
}

const sayHi = (name) => `Hi ${name}!`;
console.log(greet("Alice"));
console.log(sayHi("Bob"));

Arrays

const fruits = ["apple", "banana", "cherry"];
fruits.push("mango");
console.log(fruits[0]);
console.log(fruits);

Objects

const person = {
  name: "John",
  age: 30,
  greet() {
    return "Hi, I'm " + this.name;
  }
};
console.log(person.greet());

Loops

for (let i = 0; i < 3; i++) {
  console.log("Loop " + i);
}

let n = 0;
while (n < 2) {
  console.log("While " + n);
  n++;
}

Promises

const fetchData = new Promise((resolve, reject) => {
  setTimeout(() => resolve("Data Loaded!"), 1000);
});

fetchData.then(data => console.log(data));

Async/Await

Async/Await makes working with promises easier and cleaner.

async function getData() {
  return "Async Data Received";
}

getData().then(console.log);

Destructuring

Unpack values from arrays or properties from objects into distinct variables.

// Array
const [a, b] = [10, 20];
console.log(a, b);

// Object
const { id, status } = { id: 1, status: 'Active' };
console.log(status);
